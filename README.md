## Dev Reference

### Test locally:

```bash
npm start
```

### Build the app:

```bash
npm run build
```

### Deploy to production:

```bash
npm run deploy
```

### Prettify the code:

```bash
./node_modules/.bin/prettier --write "src/**/*.{js,jsx,ts,tsx,json,css,scss,md}"
```
